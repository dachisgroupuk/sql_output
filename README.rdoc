= sql_output

Gem to print the SQL queries generated by ActiveRecord in the Rails console

== Installation

  sudo gem install sql_output

== Usage

First, require the gem either in the rails console or add it as a dependency.

To start printing the output:

  SqlOutput.start

To stop it:

  SqlOutput.stop
